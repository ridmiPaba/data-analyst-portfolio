## 📁 Projects

### 📦 Day 1 — Superstore Sales Analysis
**Folder:** `Day1_Superstore_Sales`

**Dataset:** [Superstore Sales Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final) — 9,994 rows of retail sales data

**Business Questions Answered:**
- Which product category makes the most sales and profit?
- Which region performs best?
- Which products are losing money?

**Tools Used:**

| Tool | What was done |
|------|--------------|
| Excel | Data cleaning, Pivot Tables, Bar Charts |
| MySQL | CREATE TABLE, LOAD DATA, GROUP BY queries |
| Python | Pandas groupby(), Matplotlib bar charts |

**Key Findings:**
- Technology is the most profitable category ($145,454 profit)
- Furniture has high sales ($742,000) but very low profit ($18,451)
- West region is the best performing region ($108,418 profit)
- Central region has surprisingly low profit despite high sales
- Cubify CubeX 3D Printer is the biggest loss-making product (-$8,879)

**Files:**
- `analysis.py` — Python analysis and charts
- `quality_check.py` — Data quality checks
- `analysis.sql` — MySQL analysis queries
- `data_load.sql` — MySQL data loading script
- `superstore_table.sql` — MySQL table creation
- `chart1_category.png` — Sales and Profit by Category
- `chart2_region.png` — Sales and Profit by Region
- `chart3_loss_products.png` — Top 10 Loss Making Products
- `Day1_Project.docx` — Full documentation with findings
